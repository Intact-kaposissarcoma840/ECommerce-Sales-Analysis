# 🛒 E-commerce Sales Analysis

## 📌 Overview
This project analyzes e-commerce sales data to extract business insights.

## 📊 Features
- Data cleaning
- Sales & profit analysis
- Top products identification
- Monthly trends
- Category & region analysis
- Data visualization

## 🛠️ Tools
- Python
- Pandas
- Matplotlib

## ▶️ How to Run

1. Open project in VS Code
2. Install dependencies:
   pip install -r requirements.txt

3. Run:
   python analysis.py

## 📁 Dataset
Place your dataset inside:
data/sales.csv

## 📈 Output
- Graphs (PNG files)
- Console insights

## 📌 Conclusion
This project helps understand sales trends and customer behavior.
