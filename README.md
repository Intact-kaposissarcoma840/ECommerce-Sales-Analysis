## 📁 How to Prepare Your Data

The app works best with a clean sales file. A CSV file is a common choice. Your file should have clear columns such as:

- order date
- product name
- quantity
- price
- customer ID
- total sales

If your file uses different column names, the app may still work if the data is easy to read. Keep one row per order or sale. Try to avoid blank rows and broken values.
